Установить зависимости:

```bash
npm install
```

Запуск в режиме разработки:

```bash
npm run dev
```

Сборка:

```bash
npm run build
```

Предпросмотр собранного проекта:

```bash
npm run preview
```

Проверка линтером:

```bash
npm run lint
```
